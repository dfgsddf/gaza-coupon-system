## كيفية الاستخدام

### 1. إضافة تأثيرات بصرية
```html
<!-- تأثير التحميل التدريجي -->
<div class="fade-in-up">محتوى</div>

<!-- تأثير البطاقة المحسنة -->
<div class="card card-hover-lift">محتوى</div>

<!-- زر مع تأثير النبض -->
<button class="btn btn-primary btn-pulse">زر</button>
```

### 2. إضافة تأثيرات التحميل
```html
<!-- عنصر مع حالة التحميل -->
<div class="loading">محتوى</div>

<!-- زر مع حالة التحميل -->
<button type="submit" class="btn btn-primary">
    <span class="spinner-border spinner-border-sm me-2"></span>
    جاري التحميل...
</button>
```

### 3. إضافة التنبيهات
```javascript
// إظهار تنبيه
showAlert('تم الحفظ بنجاح!', 'success', 5000);

// إظهار تنبيه خطأ
showAlert('حدث خطأ ما!', 'danger', 5000);
```
